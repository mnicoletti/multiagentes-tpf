# Concentración

## Por posición
Una posición individual con peso elevado sobre total_ars (ej. >25–30%) es concentración explícita. Debe nombrarse en la radiografía con el peso **calculado**, no inventado.

## Por cluster
Varias posiciones chicas del mismo driver pueden sumar una concentración **oculta** mayor que cualquier ticker suelto. El cluster materializado revela ese riesgo.

## Mitigaciones (orientativas, no órdenes)
- Reducir el ticker dominante solo si no está restringido.
- Diversificar hacia drivers no correlacionados.
- Usar capital nuevo para rebalancear sin vender lo restringido.

El Planificador (F5) combina esto con restricciones duras; este doc solo fija vocabulario.
